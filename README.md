This is a complete Language Translation web application built using Python and Streamlit. It was developed as **Task 1** for the **Artificial Intelligence Internship at CodeAlpha**.

## 🚀 Live Demo
**Check out the live web app here:** [Language Translation Tool](https://translationtool-dyptis2musno2yfqtdpfej.streamlit.app/)

---

## ✨ Features
* **Multi-Language Support:** Translates text between multiple prominent languages including English, Hindi, French, German, Spanish, Japanese, Chinese, Arabic, and Russian.
* **Auto-Detect Language:** Users can choose to automatically detect the source language.
* **Text-to-Speech (TTS):** Generates playable audio of the translated text so users can hear the pronunciation.
* **Quick Copy:** Provides a formatted code block to quickly copy the translated text.
* **Interactive UI:** A clean, easy-to-use web interface powered by Streamlit.

---

## 🛠️ Technologies Used
* **[Python](https://www.python.org/)** - Core programming language.
* **[Streamlit](https://streamlit.io/)** - For building the interactive web user interface.
* **[deep-translator](https://pypi.org/project/deep-translator/)** - For connecting to the Google Translate API to perform text translations.
* **[gTTS (Google Text-to-Speech)](https://pypi.org/project/gTTS/)** - For converting the translated text into spoken audio.
